# Jokenpô em Java

Este é um projeto simples de **Jokenpô (Pedra, Papel e Tesoura)** desenvolvido em **Java**.  
O objetivo é praticar conceitos básicos de programação como:  

- Entrada de dados com `Scanner`  
- Estruturas condicionais (`if` e `switch`)  
- Comparação de strings (`equalsIgnoreCase`)  
- Geração de números aleatórios (`Math.random()`)  

---

## Como funciona

1. O programa exibe as opções: Pedra, Papel ou Tesoura.  
2. O jogador digita sua escolha.  
3. O computador faz uma escolha aleatória.  
4. O programa compara as escolhas e mostra o resultado: vitória, derrota ou empate.
